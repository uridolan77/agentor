# ADR-001 — Agentor is an Agent Runtime

## Status

Accepted

## Decision

Agentor is a deterministic, observable, policy-governed runtime for executing agents, steps, tools, skills, and evaluable runs.

Agentor is not a chatbot framework, RAG product, model gateway, or canonical knowledge engine.

## Consequences

Agentor owns execution traces and run manifests.

Agentor does not own canonical knowledge.
